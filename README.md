# The Vanishing Arctic

An interactive data storytelling project exploring Arctic climate change through data visualization and environmental analysis.

## Live Site

🌐 https://dayoungin99.github.io/the-vanishing-arctic/

## Overview

The Vanishing Arctic visualizes long-term Arctic temperature trends and environmental changes using historical climate datasets.

The project combines interactive visualizations, storytelling, and data analysis to help users understand the impact of climate change on Arctic regions over time.

## Features

- Interactive temperature anomaly visualization
- Historical Arctic climate data exploration
- Regional temperature trend analysis
- Environmental storytelling through web-based visualization
- Responsive browser experience

## Technologies

- HTML
- CSS
- JavaScript
- JSON
- Climate & temperature datasets
